## 📖 About

Welcome to the **JavaScript DSA Bootcamp**! This repository is designed to help you learn **Data Structures and Algorithms** from the ground up using modern JavaScript (ES6+).

Whether you're a beginner or preparing for coding interviews, this repository provides a structured roadmap with easy-to-understand explanations, optimized solutions, and practical coding exercises.

The goal is not just to solve problems but to understand the logic, improve problem-solving skills, and write efficient code.

# ▶️ Run the Code

Clone the repository:

```bash
git clone https://github.com/your-username/JavaScript-DSA-Bootcamp.git
```

Move into the project:

```bash
cd JavaScript-DSA-Bootcamp
```

Run a JavaScript file:

```bash
node filename.js
```
